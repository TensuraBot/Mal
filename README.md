## 🤖 About App:
In This Web-App you can see the next features:
- Most Popular Animes (Top 10)
-  Upcoming Animes (Top 10)
-  Tops
-  Search Anime by Name
-  View Anime Details, Episodes, Video, pictures, genres, etc.
